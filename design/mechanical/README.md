# mechanical

3D design exports for all physical parts of the system. Licensed under CERN-OHL-S v2.

Each subdirectory contains a `.step` (neutral CAD exchange) and a `.stl` (3D-printing / visualization) export for the corresponding part.

## Parts

| Folder | Description |
| --- | --- |
| [`arduino_shield/`](arduino_shield/) | Mechanical body / footprint model of the Arduino shield PCB. |
| [`bottom_enclosure/`](bottom_enclosure/) | Bottom half of the device enclosure. |
| [`leds_board/`](leds_board/) | Mechanical body / footprint model of the LED driver PCB. |
| [`microfluidic_device/`](microfluidic_device/) | Microfluidic chip body. |
| [`top_enclosure/`](top_enclosure/) | Top half of the device enclosure. |
| [`tsl2561_sensor/`](tsl2561_sensor/) | Housing / mounting bracket for the TSL2561 light-to-digital sensor. |
| [`velocity_sensor/`](velocity_sensor/) | Mechanical body / footprint model of the velocity sensor PCB. |
